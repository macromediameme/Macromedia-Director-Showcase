IMoaDrSound


Interface ID: IID_IMoaDrSound
Pointer type: PIMoaDrSound
Inheritance: IMoaUnknown
Header file: driservc.h

Description

This callback interface provides platform-independent access to the sound mixing capabilities of Director.
On the Macintosh, up to 8 sound channels are available, each of which can simultaneously play sounds independent of the others. On Windows, up to 4 sound channels are available for playing independent sounds.
Sound channel numbering is 1-based.

GetFreeSoundChannel()

MoaUlong FAR * piChannel))

Parameters

This
Pointer to the IMoaDrSound interface

piChannel)
Sound channel number

Returns

MoaError

Description

Call this method to get the next available sound channel. This method works from the highest-numbered sound channel down.

GetSoundChannelCount()

Syntax

GetSoundChannelCount(PIMoaDrSound This,
MoaUlong FAR * pCount)

Parameters

This
Pointer to the IMoaDrSound interface

pCount
Number of sound channels available

Returns

MoaError

Description

Call this method to determine the number of sound channels available. The value returned is platform-dependent: there are 8 sound channels available on the Macintosh and 4 on Windows.

PlaySound()

Syntax

PlaySound(PIMoaDrSound This,
MoaUlong iChannel,
PIMoaStream pSoundStream,
ConstPMoaChar psSoundFormat,
PMoaMmValue pProxyChannel)

Parameters

This
Pointer to the IMoaDrSound interface

iChannel
Sound channel number

pSoundStream
Stream containing sound data

psSoundFormat
String representing sound data format

pProxyChannel
Sprite channel playing the sound

Returns

MoaError

Description

Plays the sound provided by pSoundStream. Playback conforms to the lifetime of the sprite referred to by pProxyChannel. If a sound is already playing in the specified channel, this sound will supercede it.
