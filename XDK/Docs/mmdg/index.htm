Director 7 Xtra Development Kit

Multimedia Developer's Guide

Introduction

This document provides information you need to develop multimedia Xtras for Director 7. It includes conceptual discussions of the types of Xtras you can create, and complete documentation of the API of the Director XDK.

In the documentation

This document is in five parts, as follows:

  • "Multimedia Xtras Overview" describes the categories of Xtras you can create using the Multimedia API.

  • "Asset Xtras" describes the general capabilities and coding requirements of transition and sprite Xtras. Discusses some general multimedia and application-specific implementation details of asset Xtras.

  • "Scripting Xtras" describes the general capabilities and coding requirements of Scripting Xtras, a cross-product implementation of the Director Lingo Xtras supported in the previous release.

  • "Tool Xtras" describes general capabilities and coding requirements of Tool Xtras, which appear in the application's Xtras menu.

  • "Multimedia Support API" discusses the Multimedia Support API provided for use by Xtras, including property support data conversion, imaging support, and user interface support.

  • "Tips, Guidelines, and Gotchas" describes some useful details to be aware of when coding Xtras for Director.

For additional information on developing Xtras for Director, see the Director Developer's Guide.
