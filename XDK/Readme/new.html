Director XDK , Director 8 XDK - additions and changes

Director 8 Xtra Development Kit

New in this XDK

The following is a list summarizing the additions and changes made to the Director XDK for the Director 8 release. Detailed documentation for all items is provided in the d8xdk_addendum.html file in the Docs folder, except where noted.

New interfaces

  • Three new interfaces for defining Property pages in the Property Inspector for Asset Xtras, IMuiPropertyPage, IMuiPropertyPageTarget, and IMuiPropertyPageInstance. To set the Registry setting for new Property page, the key kMuiDictKey_PropPageEntry has also been added to the Registry Dict. Use the information in the View Description List File Syntax section to define Property Inspector controls for Xtras that you create.
  • New IMoaDrSound2 interface for Director 8, that includes all of the methods associated with IMoaDrSound interface (Director 5, 6, and 7) and a new PauseSound method.
  • IMoaDrPreferenceAccess This interface provides access to Director's preference file, which allows developers to add custom preferences (at authoring time only) that Xtras can access at any time.
  • New Notification State event ID event notification ID for the IMoaNotification interface.
  • IMoaDrMovieStage This interface provides access to new stage-related functions associated with a movie.

 

New features and fixes

  • Director 8 allows developers to create and customize tabs in the Property Inspector by editing (or adding) the VDL (View Description List) files in the props folder. Use the information in View Description List File Syntax section along with the information in the Property page interfaces section to define Property pages for Xtras that you create.
  • You can use a Castmember GetMedia call to get either a PICT (on Mac) or a DIB (on Win). If you're querying a 32-bit member, you'll get back the alpha channel in the data. You can manipulate the image, and then set it back using SetMedia. This list item is not discussed in the newinterfaces.html document. 
  • The IMoaMixAgentInfo::GetAgentFlags method can now check for the Hidden flag, as well as the HasUI flag. This list item is not discussed in the newinterfaces.html document.
  • The filename parameter for the IMoaDrPlayer interface now takes a constant value. It used to take a non-constant value. This list item is not discussed in the newinterfaces.html document.
  • The compiler macros for compiling Xtras have changed for the moaxtra.h file. This file now uses Visual C++ 6, instead of Visual C++ 5. This list item is not discussed in this document. The muiInit.h file has been updated with a new comment. This list item is not discussed in the newinterfaces.html document.
  • IMoaDrMovieCursor The description in the Director 7 XDK documentation is incorrect. The correct description is "Allows Xtras to set animated cursors or static cursors on Sprites or the Stage." This list item is not discussed in the newinterfaces.html document.
  • The compiler macros for compiling Xtras have changed for the moaxtra.h file. This file now uses Visual C++ 6, instead of Visual C++ 5. This list item is not discussed in this document. The muiInit.h file has been updated with a new comment. This list item is not discussed in the newinterfaces.html document.
  • The projects in the Examples folder have been updated to work with Metrowerks CodeWarrior 5 (Mac) and Visual C++ (Win). This list item is not discussed in the newinterfaces.html document.
